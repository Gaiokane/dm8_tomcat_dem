用户管理

  双击主页左侧导航树的用户管理节点可以打开用户管理面板。该页面用于管理用户，系统管理员可以在该页面上添加、编辑、删除用户。如果发现有用户非安全登录还可以临时禁用该用户。如下图所示:

功能按钮列表
按钮 说明
刷新 刷新用户列表。
添加 添加用户。点击打开用户添加对话框。
编辑 编辑用户。选中一个用户，点击按钮打开用户编辑对话框。
删除 选中一个或多个要删除的用户点击按钮可以删除选中的用户。
对用户列表进行登录名或真实姓名的过滤。搜索输入框中输入内容，敲下回车或者点击。

用户信息列表

  表中为系统中所有用户的信息列表。

字段 说明
登录名

用户的登录名。
状态 用户是否被禁用。表示用户可以正常使用，表示用户被禁用，被禁用的用户无法登录系统。
真实名 用户的真实姓名。
角色 用户所属角色。系统的角色类型详见角色管理面板。
邮箱地址 用户邮箱地址。用于告警邮件通知。
手机号 用户手机号。用户告警短信通知。
添加时间 添加用户的时间。

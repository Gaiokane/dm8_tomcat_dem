自定义SQL监控管理

  若已有的监控项无法满足需求，用户可以自定义sql语句进行监控。但是若想用曲线图显示监控历史信息或者想对自定义sql添加告警则要求sql语句是查询语句且查询结果为数值类型。单实例的自定义SQL监控管理入口在自定义SQL监控面板的功能按钮添加/删除，而集群的自定义SQL监控管理入口在数据库监控面板中的集群分组的操作下拉菜单中的自定义SQL监控管理。对于集群的自定义SQL监控添加，会同时在集群的所有站点上添加；同样的删除时也会删除所有站点上的自定义SQL监控。管理对话框如下图所示:
